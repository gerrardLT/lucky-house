## 九、生成注意事项

1. **尺寸要求**：
   - Hero 图：最终输出 1920×1080 或更大
   - 房型/设施图：最终输出 1200×900
   - 图标：64×64 PNG（后转 SVG）
   - Logo：输出 1024×1024 后裁切
   - OG Image：1200×630

2. **后处理建议**：
   - 所有照片类图片导出 WebP 格式，质量 80%
   - Hero 图控制在 300KB 以内
   - 图标 PNG 转 SVG 推荐使用 vectorizer.ai 或 Figma auto-trace
   - Logo 最终应为 SVG 矢量格式

3. **命名规范**：
   ```
   hero-01-family-pet-walkway.webp
   room-standard-interior-01.webp
   room-pet-friendly-interior-01.webp
   facility-onsen-outdoor-01.webp
   facility-dog-run-01.webp
   icon-onsen.svg
   icon-pet-friendly.svg
   logo-main.svg
   logo-favicon.svg
   og-default.webp
   ```
