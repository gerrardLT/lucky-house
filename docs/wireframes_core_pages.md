## 七、线框图标注约定

| 符号 | 含义 |
|------|------|
| `[文字]` | 按钮 / 输入框 / 可点击元素 |
| `(○)` `(●)` | 单选框（未选/已选） |
| `☐` `☑` | 复选框（未选/已选） |
| `[▼]` | 下拉选择器 |
| `[📅]` | 日期选择器 |
| `──` | 分割线 / 区域分隔 |
| `┌──┐` | 卡片 / 区块边界 |
| `[图片]` | 图片占位区域 |
| `*` | 必填字段标记 |
| `→` | 步骤引导 / 链接方向 |
